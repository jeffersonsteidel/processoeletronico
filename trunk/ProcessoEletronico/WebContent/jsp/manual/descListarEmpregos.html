LISTAR EMPREGOS

*É necessário permissão de Administrador para visualizar esta tela.

Dois filtros são exibidos na tela (Siape do Servidor e Nome do Servidor). O Administrador insere algum dado no filtro que deseja e o programa pesquisará baseando-se no critério fornecido. Ao clicar no botão Pesquisar , o sistema exibirá uma tabela com todos os empregos que estão relacionados com o(s) filtro(s) informado(s).

Campo Descrição Obrigatório
Siape do Servidor Siape do servidor que está relacionado com o emprego que você deseja buscar NÃO
Nome do Servidor Nome do servidor que está relacionado com o emprego que você deseja buscar NÃO

* Caso não exista registro com os dados informados aparecerá a mensagem: "Nenhum registro para o filtro informado!"
* Caso nenhum campo seja preenchido, o sistema listará todos os empregos existentes.

O resultado da pesquisa é uma tabela com os seguintes dados:
-SIAPE
-Nome do Servidor
-Empresa
-Cargo
-Data de Admissão
-Data de Saída

Ao lado direito de cada emprego presente na tabela há o campo Visualizar que é um link para uma página onde se pode visualizar todas as informações cadastradas a respeito deste emprego.
