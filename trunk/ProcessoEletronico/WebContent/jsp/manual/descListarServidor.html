LISTAR SERVIDORES


Módulo Servidor | Listar Servidores
Perfil Administrador
Última alteração 10/12/2010
Responsável Jefferson Steidel dos Santos
Contato jefferson.steidel@ifpr.edu.br

Objetivo:
Listar todos os servidores do Sistema. Nesta tela há alguns filtros (Siape, Nome, Lotação, CPF, Cargo e Situação) que podem ser preenchidos pelo Administrador. Conforme estes filtros são preenchidos, o sistema lista automaticamente os servidores que possuem os dados correspondentes aos filtros inseridos.

Procedimento:
1. Preencher os seguintes campos:(opcional)
Descrição Tipo Obrigatório
Siape Siape do servidor que você deseja buscar Numérico NÃO
Nome Nome do servidor que você deseja buscar Alfabético NÃO
Lotação Lotação do servidor que você deseja buscar Alfabético NÃO
CPF CPF do servidor que você deseja buscar Numérico NÃO
Cargo Cargo do servidor que você deseja buscar Alfabético NÃO
Situação Funcional Situação Funcional do servidor que você deseja buscar Alfabético NÃO


Resultado esperado:
Caso não ocorra nenhum erro, o sistema listará todos os servidores. Caso algum filtro sejá informado, o sistema informará apenas os servidores que correspondam àquele filtro. os dados retornados são:
- Siape - Siape do Servidor
- Nome - Nome do Servidor
- Lotação - Lotação do Servidor
- CPF - CPF do Servidor
- Cargo - Cargo do Servidor
- Situação Funcional - Situação Funcional do Servidor
- Editar - Link para pagina de Edição dos Dados do Servidor
- Ativo - Icone que contém: 1)Um icone com seta verde para servidores que atualmente trabalham no IFPR. 2)Um icone com seta vermelha para servidores que sairam do IFPR.
- Validado - Alerta se os dados foram validados, ou não, pela PROGEPE.

Possíveis erros:
Mensagem Motivo
Erro ao comunicar com o servidor! Ocorreu alguma falha durante a realização da operação!
