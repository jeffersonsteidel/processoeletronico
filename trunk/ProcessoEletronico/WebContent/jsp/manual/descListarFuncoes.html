Pesquisar Servidor

*É necessário permissão de Administrador para visualizar esta tela.

Alguns filtros são exibidos na tela (Tipo Função, Função, Siape, Local de Exercício e Atual).


Campo Descrição Obrigatório
Tipo Função Tipo da Função que você deseja buscar NÃO
Função Função que você deseja buscar NÃO
Siape Siape do servidor que exerce a função que você deseja buscar NÃO
Local Execício Campus que você deseja buscar NÃO

O Administrador insere algum dado no filtro que deseja e o programa pesquisará as funções baseando-se no critério fornecido. Ao clicar no botão pesquisar, o sistema exibirá uma tabela com todos os servidores relacionados com o(s) filtro(s) informado(s).

* Caso não exista registro com os dados informados aparecerá a mensagem: "Nenhum registro para o filtro informado!"
* Caso nenhum campo seja preenchido, o sistema listará todos as funções existentes.

O resultado da pesquisa é uma tabela com os seguintes dados:
-Tipo Função
-Função
-Nome do Servidor
-Local Exercício
-Data Entrada
-Data Saída
-Ato de Criação

Ao lado direito de cada servidor presente na tabela há um campo: Editar .
* O campo Editar é um link para uma pagina de Edição das Funções.
