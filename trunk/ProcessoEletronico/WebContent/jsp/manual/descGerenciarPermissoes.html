GERENCIAR PERMISSÕES

Módulo Administração | Gerenciar Permissões
Perfil Gerente
Última alteração 21/02/2011
Responsável Jefferson Steidel dos Santos
Contato jefferson.steidel@ifpr.edu.br

Objetivo:
Alterar permissões de um servidor no sistema.

Procedimento:
1. Preencher os seguintes campos:
Descrição Tipo Obrigatório
SIAPE Número SIAPE do servidor Numérico SIM
2. Selecionar as permissões do servidor para alteração.
3. Clicar em salvar.

Resultado esperado:
Caso não ocorra nenhum erro, o sistema salva os dados no banco e exibirá a mensagem "Item alterado com sucesso!".

Possíveis erros:
Mensagem Motivo
Siape inexistente! Não existe um servidor cadastrado no sistema com este número SIAPE
Este servidor está desativado! O servidor relacionado ao número SIAPE informado está inativo e teve seu acesso ao sistema cancelado!
Erro ao comunicar com o servidor! Ocorreu alguma falha durante a realização da operação!
