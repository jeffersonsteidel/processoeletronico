Manual Do Usuário

GERENCIAR PERMISSÕES



Permissões:
Usuários gerentes do sistema.

Objetivo da tela:
Permitir alterações nas permissões de um servidor ao sistema.

Fluxo de preenchimento:
Deve-se informar um siape. Se o siape informado se referir a um servidor que possui acesso ao sistema, serão carregados campos para marcar as permissões que o servidor terá dentro do sistema.

Resultado:
O servidor teve suas permissões alteradas com sucesso.

Possíveis erros:
Mensagem Motivo
Siape inexistente! Não existe um servidor cadastrado no sistema com este numero SIAPE
Este servidor está desativado! O servidor relacionado a este numero SIAPE está inativo e teve seu acesso ao sistema cancelado
Erro ao comunicar com o servidor! Ocorreu alguma falha durante a realização da opração
