LISTA DE LOTAÇÕES

Módulo Administração | Listar Lotações
Perfil Administrador
Última alteração 10/12/2010
Responsável Jefferson Steidel dos Santos
Contato jefferson.steidel@ifpr.edu.br

Objetivo:
Utilizada para listar as lotações cadastradas no sistema.

Procedimento:
1. Preencher os seguintes campos:
Descrição
Código Código do Cargo
Campus Nome do Campus
Diretor Administrativo Nome do Diretor Administrativo
Diretor Geral Nome do Diretor Geral
Endereço Endereço do Campus

Resultado esperado:
Caso não ocorra nenhum erro, o sistema listará todos as lotações. Caso algum filtro sejá informado, o sistema informará apenas as lotações que correspondam àquele filtro. os dados retornados são:
- Código
- Campus
- Diretor Administrativo
- Diretor Geral
- Endereço

Ao clicar em EDITAR abrirá uma tela para edição da Lotação.

Possíveis erros:
Mensagem Motivo
Nenhum registro para o filtro informado! Não há servidor que sejá compatível com os filtros selecionados.
Erro ao comunicar com o servidor! Ocorreu alguma falha durante a realização da operação!
