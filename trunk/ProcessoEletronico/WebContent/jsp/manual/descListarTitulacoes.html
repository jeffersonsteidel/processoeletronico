LISTAR TITULAÇÕES

*É necessário permissão de Administrador para visualizar esta tela.

Quatro filtros são exibidos na tela (Siape, Nome, Titulação e Área de Conhecimento). O Administrador insere algum dado no filtro que deseja e o programa pesquisará baseando-se no critério fornecido.

Campo Descrição Obrigatório
Siape Siape do servidor que está relacionado com a titulação que você deseja buscar NÃO
Nome Nome do servidor que está relacionado com a titulação que você deseja buscar NÃO
Titulação Selecione o tipo de titulação que você deseja buscar NÃO
Área de Conhecimento Selecione a Área de Conhecimento que está relacionada com a titulação que você deseja buscar NÃO
Validados Situação dos servidores que aparecereção na lista
TODOS - listar todos
SIM - listar apenas titulações que já foram validas pela PROGEPE
NÃO - listar apenas titulações que NÃO foram validas pela PROGEPE		 NÃO
Situação Situação dos servidores que aparecereção na lista
TODOS - listar todos
ATIVOS - listar apenas os que ainda trabalham no IFPR
INATIVOS - listar apenas os que não trabalham mais no IFPR		 NÃO

* Caso não exista registro com os dados informados aparecerá a mensagem: "Nenhum registro para o filtro informado!"
* Caso nenhum campo seja preenchido, o sistema listará todos as titulações existentes.

Ao clicar no botão Pesquisar , o sistema exibirá uma tabela com todos as titulações que estão relacionados com o(s) filtro(s) informado(s).
O resultado da pesquisa é uma tabela com os seguintes dados:
-SIAPE
-Nome do Servidor
-Titulação
-Área de Conhecimento

Ao lado direito de cada titulação presente na tabela há o campo Visualizar que é um link para uma página onde se pode visualizar todas as informações cadastradas a respeito desta titulação.
