Manual Do Usuário

Atualizar Menus Dados / Cadastrar Servidor


Etapa do sistema responsável pela atualização dos dados do servidor logado ou pelo cadastramento de um novo servidor*.
É composta por 6 abas (Dados Pessoais, Dados Funcionais, Documentos, Contato, Conta Bancária e Salvar).
*É necessário permissão de administrador para cadastrar um novo servidor.

Campo Descrição
