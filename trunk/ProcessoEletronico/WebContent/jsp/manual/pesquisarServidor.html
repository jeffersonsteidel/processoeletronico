Pesquisar Servidor

Esta é tela para pesquisa de servidores. É necessário permissão de administrador para vizualizar esta tela.

Campo Descrição Obrigatório
Siape Siape do servidor que você deseja buscar NÃO
Nome Nome do servidor que você deseja buscar NÃO
Lotação Lotação do servidor que você deseja buscar NÃO
Cargo Cargo do servidor que você deseja buscar NÃO

*Caso nenhum campo seja preenchido ou não exista registro com os dados informados aparecerá a mensagem: "Nenhum registro para o filtro informado!" *O campo Editar é um link para a pagina de Edição dos Dados do Servidor.
