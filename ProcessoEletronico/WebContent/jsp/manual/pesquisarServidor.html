Pesquisar Servidor

Alguns filtros são exibidos na tela (Siape, Nome, Lotação e Cargo). O Administrador insere algum dado no filtro que desejar para que o programa faça a pesquisa baseando-se em algum critério. Ao clicar no botão pesquisar, o sistema exibe em uma tabela todos os servidores que possuem dados correspondentes aos filtros inseridos.
É necessário permissão de administrador para visualizar esta tela.

Campo Descrição Obrigatório
Siape Siape do servidor que você deseja buscar NÃO
Nome Nome do servidor que você deseja buscar NÃO
Lotação Lotação do servidor que você deseja buscar NÃO
Cargo Cargo do servidor que você deseja buscar NÃO

* Caso não exista registro com os dados informados aparecerá a mensagem: "Nenhum registro para o filtro informado!"
* Caso nenhum campo seja preenchido, o sistema listará todos os servidores existentes.

Ao lado direito de cada servidor presente na tabela há um campo: Editar .
*O campo Editar é um link para uma pagina de Edição dos Dados do Servidor.
