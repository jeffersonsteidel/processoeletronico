PESQUISAR CÔNJUGES

Módulo Servidor | Pesquisar Cônjuges
Perfil Administrador
Última alteração 14/02/2011
Responsável Jefferson Steidel dos Santos
Contato jefferson.steidel@ifpr.edu.br

Objetivo:
Exibir todos oscônjuges que estão relacionados com o(s) filtro(s) informado(s).

Procedimento:
1. Preencher os seguintes campos:

*É necessário permissão de Administrador para visualizar esta tela.
Dois filtros são exibidos na tela (Siape e Nome). O Administrador insere algum dado no filtro que deseja e o programa pesquisará baseando-se no critério fornecido. informado(s).

Descrição Obrigatório
Siape do Servidor Siape do servidor que está relacionado com o cônjuge que você deseja buscar NÃO
Nome do Servidor Nome do servidor que está relacionado com o cônjuge que você deseja buscar NÃO
Situação Situação dos servidores que aparecerão na lista
TODOS - listar todos
ATIVOS - listar apenas os que ainda trabalham no IFPR
INATIVOS - listar apenas os que não trabalham mais no IFPR		 NÃO
Validados Validação feita pela PROGEPE
TODOS - listar todos
ATIVOS - listar apenas os cônjuges aprovados pela PROGEPE
INATIVOS - listar apenas os cônjuges que não receberam aprovação da PROGEPE		 NÃO

* Caso não exista registro com os dados informados aparecerá a mensagem: "Nenhum registro para o filtro informado!"
* Caso nenhum campo seja preenchido, o sistema listará todos os cônjuges existentes.

2. Clicar em pesquisar.

Resultado esperado:
O resultado da pesquisa é uma tabela com os seguintes dados:
-SIAPE
-Nome do Servidor
-Nome do Cônjuge
-Cônjuge Atual

Ao lado direito de cada cônjuge presente na tabela há um campo: Editar .
* O campo Editar é um link para uma pagina de Edição dos Dados do Cônjuge.

Possíveis erros:
Mensagem Motivo
Nenhum registro para o filtro informado! Não há servidor que sejá compatível com os filtros selecionados.
Erro ao comunicar com o servidor! Ocorreu alguma falha durante a realização da operação!
