LISTA DE CARGO

Módulo Administração | Listar Cargos
Perfil Administrador
Última alteração 10/12/2010
Responsável Jefferson Steidel dos Santos
Contato jefferson.steidel@ifpr.edu.br

Objetivo:
Utilizada para listar os cargos cadastradas no sistema.

Procedimento:
1. Preencher os seguintes campos:
Descrição
Código Código do Cargo
Cargo Nome do Cargo
Classe Classe a qual o cargo se enquadra
Editar Editar Cargo

Resultado esperado:
Caso não ocorra nenhum erro, o sistema listará todos os cargos. Caso algum filtro sejá informado, o sistema informará apenas os cargos que correspondam àquele filtro. os dados retornados são:
- Código
- Cargo
- Classe

Ao clicar em EDITAR abrirá uma tela para edição do Cargo.

Possíveis erros:
Mensagem Motivo
Nenhum registro para o filtro informado! Não há servidor que sejá compatível com os filtros selecionados.
Erro ao comunicar com o servidor! Ocorreu alguma falha durante a realização da operação!
