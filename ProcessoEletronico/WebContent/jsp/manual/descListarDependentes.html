PESQUISAR DEPENDENTES

Módulo Servidor | Pesquisar Dependentes
Perfil Administrador
Última alteração 14/02/2011
Responsável Jefferson Steidel dos Santos
Contato jefferson.steidel@ifpr.edu.br

Objetivo:
Exibir todos os dependentes que estão relacionados com o(s) filtro(s) informado(s).

Procedimento:
1. Preencher os seguintes campos:

*É necessário permissão de Administrador para visualizar esta tela.
Quatro filtros são exibidos na tela (Siape, Servidor, Dependente e Parentesco). O Administrador insere algum dado no filtro que deseja e o programa pesquisará baseando-se no critério fornecido.
Descrição Obrigatório
Siape Siape do servidor que está relacionado com o dependente que você deseja buscar NÃO
Nome Nome do servidor que está relacionado com o dependente que você deseja buscar NÃO
Dependente Nome do dependente que você deseja buscar NÃO
Parentesco Grau de parentesco entre o servidor logado e o dependente que você deseja buscar NÃO
Validados Filtro para verificar os dependentes validados ou não
TODOS - listar todos
SIM - listar apenas os dependentes já validados pela PROGEPE
NÃO - listar apenas os dependentes que NÃO foram validados pela PROGEPE		 NÃO
Situação Situação dos servidores que aparecereção na lista
TODOS - listar todos
ATIVOS - listar apenas os que ainda trabalham no IFPR
INATIVOS - listar apenas os que não trabalham mais no IFPR		 NÃO

* Caso não exista registro com os dados informados aparecerá a mensagem: "Nenhum registro para o filtro informado!"
* Caso nenhum campo seja preenchido, o sistema listará todos os dependentes existentes.

2. Clicar em pesquisar.

Resultado esperado:
O resultado da pesquisa é uma tabela com os seguintes dados:
-SIAPE
-Nome do Servidor
-Nome do Dependente
-Grau de Parentesco

Ao lado direito de cada dependente presente na tabela há o campo Visualizar que é um link para uma página onde se pode visualizar todas as informações cadastradas a respeito deste dependente.

Possíveis erros:
Mensagem Motivo
Nenhum registro para o filtro informado! Não há servidor que sejá compatível com os filtros selecionados.
Erro ao comunicar com o servidor! Ocorreu alguma falha durante a realização da operação!
