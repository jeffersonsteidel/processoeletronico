Objetivo:
O sistema exibirá uma tabela com todos os empregos que estão relacionados com o(s) filtro(s) informado(s)

Procedimento:
1. Preencher os seguintes campos:
Descrição Obrigatório
Tipo Função Tipo da Função que você deseja buscar NÃO
Função Função que você deseja buscar NÃO
Siape Siape do servidor que exerce a função que você deseja buscar NÃO
Local Execício Campus que você deseja buscar NÃO
Atual Buscar apenas as Funções exercidas atualmente NÃO

2. Clicar em Pesquisar.

Resultado esperado:
Caso não ocorra nenhum erro, o sistema listará todos os servidores. Caso algum filtro sejá informado, o sistema informará apenas os servidores que correspondam àquele filtro. os dados retornados são:
-Tipo Função
-Função
-Nome do Servidor
-Local Exercício
-Data Entrada
-Data Saída
-Ato de Criação

*Editar é um link para uma pagina de Edição das Funções.

Possíveis erros:
Mensagem Motivo
Nenhum registro para o filtro informado! Não há servidor que sejá compatível com os filtros selecionados.
Erro ao comunicar com o servidor! Ocorreu alguma falha durante a realização da operação!
