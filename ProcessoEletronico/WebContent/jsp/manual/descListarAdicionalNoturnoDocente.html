LISTAR ADICIONAL NOTURNO - DOCENTE



*É necessário permissão de Diretor para visualizar esta tela.

Esta tela permite o diretor validar quais os servidores trabalharam após as 22 horas.
Após validar o mesmo deverá encaminhar a lista para a PROGEPE clicando no botão "Encaminhar para a Progepe".

Conteúdo da Lista
Campo Descrição
Nome Nome do Docente
Data Data que o docente ministrou aulas após as 22:00 horas
Dia da Semana Dia da Semana que o docente ministrou aulas após as 22:00 horas
Horário Horário que o docente ministrou as aulas (o horário deve ser compreendido entre 22:00 e 05:00 horas)
Curso - Turma - Matéria Curso, turma e Matéria que o docente ministrou a aula naquele intervalo de tempo.
Aprovação Deve ser checado caso as informações estejam corretas, os adicionais não checados não serão validados pela PROGEPE.
