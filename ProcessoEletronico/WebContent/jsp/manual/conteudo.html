Manual Do Usuário

Objetivos

O Processo Verde é um sistema desenvolvido pela PROGEPE com objetivo de dar mais agilidade no atendimento a solicitações realizadas pelos servidores.

Com esse sistema é possível que o servidor possa acompanhar suas solicitações e

Usuários

- Administradores
Tem permissão para realizar qualquer operação no sistema.

- Aprovadores
Tem permissão para realizar aprovação de solicitações.

- Servidores
Tem permissão para realizar solicitações.
