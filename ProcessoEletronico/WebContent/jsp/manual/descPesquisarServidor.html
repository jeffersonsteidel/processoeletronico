PESQUISAR SERVIDOR

Módulo Servidor | Pesquisar Servidor
Perfil Administrador
Última alteração 10/12/2010
Responsável Jefferson Steidel dos Santos
Contato jefferson.steidel@ifpr.edu.br

Objetivo:
Pesquisar servidores de acordo com os filtros informados.

Procedimento:
1. Preencher os seguintes campos: (opcional)
Descrição Tipo Obrigatório
Siape Siape do servidor que você deseja buscar Numérico NÃO
Nome Nome do servidor que você deseja buscar Alfabético NÃO
Lotação Lotação do servidor que você deseja buscar Alfabético NÃO
Cargo Cargo do servidor que você deseja buscar Alfabético NÃO
Situação Situação dos servidores que aparecereção na lista
TODOS - listar todos
ATIVOS - listar apenas os que ainda trabalham no IFPR
INATIVOS - listar apenas os que não trabalham mais no IFPR		 Alfabético NÃO

2. Clicar em pesquisar.

Resultado esperado:
Caso nenhum campo seja preenchido, o sistema listará todos os servidores existentes.
Caso não ocorra nenhum erro, o sistema listará os servidores que possuem os requisitos dos filtros.
Valores retornados:
-Siape
- Nome
- Lotação
- CPF
- Cargo
- Situação Funcional

Ao lado direito de cada registro retornado há um campo Editar. O campo Editar é um link para uma pagina de Atualizar Meus Dados com os dados do Servidor selecionado preenchido.

Possíveis erros:
Mensagem Motivo
Nenhum registro para o filtro informado! Não há servidor que sejá compatível com os filtros selecionados.
Erro ao comunicar com o servidor! Ocorreu alguma falha durante a realização da operação!
