Alterar Senha

Esta é a tela de alteração da senha de login. Para alterar sua senha, é preciso preencher os campos.

Campo Descrição Obrigatório
Senha Atual Senha que você usa atualmente para logar SIM
Nova Senha Senha que você deseja usar SIM
Confirmar Nova Senha* Valida se a nova senha foi escrita corretamente SIM

*Esta senha deve ser igual a Nova Senha.


Caso a Senha Atual digitada seja inválida, o sistema indicará o seguinte erro: Senha Atual incorreta!;
Caso a Nova Senha e a Confirmar Nova Senha sejam incompativeis, o sistema indicará o seguinte erro: Confirmar Nova Senha diferente!;
