LISTAR SERVIDORES

Todos os servidores são listados de acordo com o siape. Nesta tela há alguns filtros (Siape, Nome, Lotação, CPF, Cargo e Situação) que podem ser preenchidos pelo Administrador. Conforme estes filtros são preenchidos, o sistema lista automaticamente os servidores que possuem os dados correspondentes aos filtros inseridos.
*É necessário permissão de administrador para visualizar esta tela.

Campo Descrição Obrigatório
Siape Siape do servidor que você deseja buscar NÃO
Nome Nome do servidor que você deseja buscar NÃO
Lotação Lotação do servidor que você deseja buscar NÃO
CPF CPF do servidor que você deseja buscar NÃO
Cargo Cargo do servidor que você deseja buscar NÃO
Situação Funcional Situação Funcional do servidor que você deseja buscar NÃO

Ao lado direito de cada servidor presente na tabela há dois campos: Editar e Validado.
*O campo Editar é um link para uma pagina de Edição dos Dados do Servidor.
*O campo Ativo exibe um icone que contém:
1)Um icone com seta verde para servidores que atualmente trabalham no IFPR.
2)Um icone com seta vermelha para servidores que sairam do IFPR.
*O campo Validado mostra um alerta se os dados foram validados, ou não, pela PROGEPE.
